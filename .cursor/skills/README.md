# Local Skills

These are the default local skills for this repo.

## Core skills

1. `scope-guard` - Use when a request risks adding features outside the current proof slice.
2. `search-first` - Use before proposing new dependencies, utilities, abstractions, or platform changes.
3. `proof-pack` - Use when implementing or validating any execution slice.
4. `docs-drift-sync` - Use after meaningful implementation changes to keep docs aligned.

## Rule

Do not add more skills unless the product genuinely needs them.
